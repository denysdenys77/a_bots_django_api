Перед запуском проекта создайте базу данных PostgreSQL, и пользователя с соответсвующими правами.
Укажите имя пользователя, его пароль, имя базы данных в файле .env
После проведения миграций, запустите комманду "python manage.py loaddata referral_links_fixtures" для автоматической загрузки исходных данных в базу.

Реферальные ссылки для входа в бот находяться в файле links.txt
